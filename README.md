# Calculadora de Faltas
## Descrição
O programa recebe a quantidade de ausências e mostra o percentual.

## Objetivo e Público-Alvo
Alertar os estudantes para que estejam cientes da quantidade de ausências não ultrapassem o limite de 25%.

## Funcionalidades
1. Mostra o percentual de faltas
2. Indica se ultrapassou ou está abaixo do limite
